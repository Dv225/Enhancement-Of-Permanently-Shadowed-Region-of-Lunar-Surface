OUR APPROACH


Technologies Used :

Language : Python
Libraries : OpenCV, PyTorch, NumPy, Tensorflow,GDAL, matplotlib, Scikit-images
Tools/Frameworks : Google Colab, QGIS, keras API
Training : Nvidia CUDA
Dataset : PRADAN

Idea of our model :

- OHRC and IIRS images are collected and preprocessed . Necessary transformations are applied on the data .​
- A shared Encoder architecture is used to extract common low-level features of both the images, which then get branched into dual outputs for
SE(Structural Embeddings) and RE(Resolution Embeddings).​
- Attention mechanism is used to focus on relevant regions of the images during encoding and decoding.
- CGAN architecture is used to generate enhanced images using both datasets.​
- Perceptual and standard embedding losses are incorporated to improve the quality of generated images.​
- Model is trained iteratively until an enhanced image is generated by updating the generator and discriminator using a combination of embedding losses,
perceptual loss, and conditional data.
- Model testing is done for the target image of the PSR of the moon.​
- Output is further denoised and enhanced using CLAHE algorithm.​
